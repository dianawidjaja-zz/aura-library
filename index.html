Aura is a component-based UI framework that provides intelligent client and server interaction.

View on Github Component Reference

Introduction

Components are the building blocks of Aura. With varying granularity, they encapsulate a modular reusable section of UI, providing comprehensive support for user interface events through component attributes and client-side controllers. Use these components out-of-the-box or extend and customize them based on your needs.

For a full list of component attributes and events, see the Reference site.

Alerts

ui:message

Display a message of varying severity level using the severity attribute. Set the closable attribute to true if displaying a closable button.

<ui:message title="Confirmation" severity="confirm">
    This is a confirmation message.
</ui:message>

<ui:message title="Information" severity="info" closable="true">
    This is a message.
</ui:message>

 <ui:message title="Warning" severity="warning">
    This is a warning.
</ui:message>

<ui:message title="Error" severity="error">
    This is an error message.
</ui:message>

Buttons

ui:button

Buttons can contain text only, image only, or both text and image. Setting labelClass="assistiveText" hides the label but maintains its accessibility.

<ui:button label="Submit"/>

<ui:button label="Search" class="img"/>
.THIS.uiButton.img {
    background:url(/img/icon/lightning16.png) no-repeat;
}
